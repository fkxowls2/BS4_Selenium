# BS4_Crawling
### BS4를 활용한 웹스크롤링
> EBS 명의 시청자 게시글을 크롤링하여 텍스트로 저장

코드 진행
1. 함수 실행(게시글 페이지 수 입력)
2. 각 게시글의 링크주소를 추출
3. 게시글 내용을 텍스트 파일로 저장
